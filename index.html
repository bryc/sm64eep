Level flags
Level flag format:
All the level flags follow the same format: bits 0:6 allocate stars, and the last bit, bit 7 controls the cannon of the previous level. This makes it possible to have 26 * 7 = 182 stars saved, even though there are only 120 available in normal gameplay.

Level format:
Every Level byte shares the same format. The first 7 bits add stars to your total, while the 8th is reserved. This makes it possible to have 26 * 7 = 182 stars saved, even though there are only 120 available in normal gameplay. The 8th bit is only known to control whether the previous level's cannon is unlocked. This makes it impossible for a cannon to be unlocked in the level bytes 0x08 and 0x24. Also, not every level actually has a cannon that can be unlocked.
